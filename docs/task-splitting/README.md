## üß† Key Concepts

### Auto-Chunker Algorithm
- **Intent Detection**: Parse task language for verbs, objects, context
- **Scope Classification**: Simple, Multi-step, or Project level
- **Micro-Step Generation**: Break into 2-5 minute actionable chunks
- **Delegation Detection**: Identify tasks suitable for AI agents

### ADHD-First Design
- **Single Focus View**: Show one micro-step at a time
- **Immediate Action**: Reduce time to first action under 60 seconds
- **Visual Clarity**: Clean, distraction-free interface
- **Dopamine Rewards**: XP and celebration for micro-step completion

### Mobile-First Approach
- **Touch Interactions**: Swipe to complete, delegate, or defer
- **Voice Input**: "Split this task" voice commands
- **Haptic Feedback**: Physical confirmation of actions
- **One-Hand Use**: Optimized for single-hand mobile operation
